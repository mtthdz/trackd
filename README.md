## Trackd ##
version 0.0.1 <br />

Trackd is a nutrition tracking app built with React, Express, NodeJS and MongoDB. <br />


### Getting Started ###

```console
install dependencies
$ npm install

starts nodemon on port 8000 (requires connection to MongoDB)
$ npm start
```
